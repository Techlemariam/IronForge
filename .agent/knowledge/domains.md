# IronForge Domain Definitions

Quick reference for domain-specific context when using `/domain-session`.

## 🎨 Frontend (`frontend`)
Next.js App Router, Tailwind v4 design system ("Chromatic Lattice"), Shadcn-style UI primitives, Framer Motion animations, Storybook, PWA, multi-platform (desktop/mobile/TV).

## 🔧 Infrastructure/DevOps (`infra`)
CI/CD pipelines, GitHub Actions, Docker, Nix, database migrations, Vercel deployment, monitoring, Sentry, build system.

## 🎮 Game Mechanics (`game`)
Titan progression, XP systems, leveling, attributes, combat calculations, dungeon mechanics, Battle Pass, PvP, guilds.

## 📋 Sprint Planning (`sprint`)
Sprint planning, roadmap management, backlog grooming, velocity tracking, prioritering.

## 🧪 Testing/QA (`qa`)
Unit tests, E2E tests, Playwright, test coverage, regression testing, stress tests.

## 🧬 Bio Integration (`bio`)
Integration with external fitness APIs (Intervals.icu, Hevy), heart rate processing, bio-buffs, wellness→game stat translation.

## 💰 Business (`business`)
Subscription models, Stripe integration, pricing, lifetime access, monetization strategy.

## 🔌 External APIs (`api`)
External API routes, authentication, third-party integrations (Spotify, Pocket Casts, etc), webhooks.

## 🧠 Meta/Workflows (`meta`)
Workflow optimization, agent configuration, development process improvements, .agent/ structure, Gemini Brotherhood roster.
